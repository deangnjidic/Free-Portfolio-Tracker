üöÄ Getting Started

Free Portfolio Tracker runs entirely in your browser. Your data is stored locally on your device ‚Äî nothing is ever sent to any server.

First-Time Setup

  • 1Open the app and click ‚öôÔ∏è Settings
  • 2Set your preferred names for Person 1 and Person 2 (e.g. your names, or "Me" / "Partner")
  • 3Paste your free Finnhub API key for live stock & crypto prices ‚Äî get one at finnhub.io/register
  • 4Optionally paste a Metals.dev API key if you hold gold, silver, or other precious metals ‚Äî free at metals.dev
  • 5Click Save Settings and start adding assets
üí°
The app ships with demo data so you can explore before entering your own holdings. Delete demo assets whenever you're ready.

  • 1Refresh prices on the Portfolio page first
  • 2Click üíæ Save Snapshot

üíæ Backup & Restore

  • 1Click Export on the Portfolio page
  • 2A portfolio_backup.json file downloads ‚Äî includes all assets, snapshots, and settings
  • 3Store this file somewhere safe (cloud storage, USB drive)

  • 1Click Import on the Portfolio page
  • 2Select your portfolio_backup.json file
  • 3Your portfolio, snapshots, and settings are fully restored
‚ö†Ô∏è
Warning: Importing replaces all current data. Export a backup first if you want to preserve your existing portfolio.
üîí Privacy & Security
  • 100% local ‚Äî All data stored in your browser's localStorage. No backend, no server, no cloud sync.
  • No portfolio analytics ‚Äî Google Analytics tracks page views only. Your holdings, symbols, and values are never transmitted.
  • API keys stay local ‚Äî Your Finnhub and Metals.dev keys are stored in localStorage only and never committed to any repository.
  • Clearing browser data ‚Äî Deleting site data or localStorage will erase your portfolio. Export a backup first.
  • Multiple portfolios ‚Äî Use different browser profiles or browsers to maintain completely separate portfolios.
üõ†Ô∏è Troubleshooting

Prices not updating

  • Check your API key is saved in Settings
  • Wait 60 seconds ‚Äî prices are cached and refreshes are rate-limited
  • Verify your key is valid at finnhub.io
  • Open DevTools (F12 ‚Üí Console) to see specific error messages

Symbol not found

  • Stocks ‚Äî Use the exact ticker (e.g. AAPL, not "Apple")
  • Crypto ‚Äî Finnhub requires exchange prefix format, e.g. BINANCE:BTCUSDT
  • Metals ‚Äî Use uppercase codes: XAU (gold), XAG (silver), XPT (platinum)
  • Use the autocomplete search when adding an asset ‚Äî it queries Finnhub directly

Data disappeared

  • Check browser data wasn't cleared ‚Äî localStorage is wiped when you clear site data
  • Make sure you're on the same browser and profile you used originally
  • Restore from your last JSON export backup
